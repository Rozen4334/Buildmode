# Buildmode
A build-mode plugin for TShock, by nyan-ko & Nova4334

## Features:

- Remove background (by moving the cave layer down to hell)
- freezetime on 12:00PM
- custom settime that ignores the server realtime

## Commands:

- /buildmode
- /bm
- /stime
- /stime (time)

## Permissions:

- buildmode.toggle
- buildmode.stime
